jock-popUp2.js 带关闭按钮的通用弹出层

1：如何使用

引入js文件jock-popUp2.js，不依赖任何其他库，不需要引入任何CSS文件。

复制默认的关闭按钮背景图片popup_close_layer.png到当前js目录，也可以自定义关闭按钮背景图，如示例。

此控制的功能只是用来弹出指定的一个DOM对象，本身并不生成弹出的层，也不处理弹出层的任何事件

2：popUp方法

			    /*显示方法*/
				popUp.show(dom, size[, options]);

					dom: 你要弹出的层对象,如 var dom = document.getElementById("#id");

					size:弹出层的宽高,如 var size = {height: 200, width: 200};

					options:遮罩，动画效果参数

					如果不提供options参数，它的默认值是
					{
						color: 'black', /*遮罩默认为黑色*/
						opacity: 0.5,   /*遮罩透明度默认为0.5*/
						alpha: true,    /*弹出时是否有透明度渐变动画，默认为true*/
						slide: true    /*弹出时是否有滑动动画，默认为true*/
						closeLayerImg = 'popup_close_layer.png';
					}
				
				/*隐藏方法*/
				popUp.hide();

示例: 

				var html = document.getElementById("contentTemplate").innerHTML;
				var pop = document.createElement("div");
					pop.innerHTML = html;
				
				var popBtn = document.getElementById("popBtn");
					
				popBtn.onclick = function(){
					popUp.show(pop, {height: 490, width: 330}, {slide: true, alpha: true, opacity: 0.8, color: '#111'});
				};
